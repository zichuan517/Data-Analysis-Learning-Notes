# 04 玩转两个变量数据

1.散点图  Scatter Plot

`sns.scatterplot(x,y)`

2.折线图  Line Plot

`sns.lineplot(x,y)`

3.柱形图  Bar Plot

`sns.barplot(x,y)`

4.计数图  Count Plot

`sns.countplot(x,y)`

5.饼图 Pie Plot

`plt.pie(['col'], labeels=['col'], autopct='%.1f%%')`
